校园数字孪生系统

项目概述

本系统基于Three.js构建的校园数字孪生交互平台，通过3D可视化技术展现校园空间布局。场景模型使用腾讯混元3D生成式AI工具生成，支持在浏览器中进行沉浸式浏览与交互。

功能说明

  • 鼠标操作：左键旋转、右键平移、滚轮缩放
  • 键盘移动：WASD键控制相机前后左右移动
  • 视角切换：支持透视、鸟瞰、侧面三种视角
  • 对象交互：点击场景中的对象可查看详细信息
  • 自动旋转：开启后场景自动环绕旋转展示

技术栈

HTML5 / CSS3 / JavaScript / Three.js / GLTFLoader
